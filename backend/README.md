# DevOps Pal Backend

### Get a terminal (ctrl+shift+~)

1. `cd backend`
2. `python3 -m venv devops_pal`
3. `devops_pal\Scripts\activate`
4. `pip install -r requirements.txt`
5. `python server.py`

### Get another terminal (ctrl+shift+~)

1. `cd backend`
2. `python3 -m venv devops_pal`
3. `devops_pal\Scripts\activate`
4. `python client_api_ollama.py`

### Both server.py and client_api_ollama.py must be running in order to whole system to be operational.
